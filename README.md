## 📁 Projects

### 🎵 [音乐多轨分离工具](https://github.com/Eruditi/music_separator_app)
- **Description**: A local audio separation desktop application using AI models to separate audio files into multiple tracks (vocals, instruments, etc.)
- **Tech Stack**: Python, PyQt5, PyTorch, Demucs, FFmpeg
- **Features**: Multi-track separation (2/4/6 tracks), real-time preview, GPU acceleration, batch processing

### 🔌 [CE MCP Plugin](https://github.com/Eruditi/CE-MCP-Plugin)
- **Description**: A Cheat Engine plugin that implements Model Context Protocol (MCP), allowing AI models to interact with Cheat Engine
- **Tech Stack**: C
- **Features**: Memory read/write, process management, AI integration

### 📹 [AI 边缘计算摄像头分析盒子](https://github.com/Eruditi/ai-box)
- **Description**: AI edge computing camera analysis system with multiple intelligent algorithms covering person violation, environmental anomalies, perimeter alert, behavior alert, full structured analysis, and face recognition
- **Tech Stack**: Python, YOLO, OpenCV
- **Features**: Multiple AI algorithms, real-time analysis, alert system
